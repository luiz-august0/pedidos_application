# pedidos_application
Aplicação web de gerenciamento de pedidos e produtos desenvolvida com ReactJS, NodeJS e banco de dados MySQL. Foram utilizadas nesta aplicação tecnologias como pdfmake para geração de relatõrios em pdf, express para desenvolvimento de apis rest, autenticação por JWT, material-ui para interface do front-end, componente ag-grid para renderização de tabelas e entre outras tecnologias.
